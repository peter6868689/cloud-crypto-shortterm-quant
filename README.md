# Cloud-Crypto-Shortterm-Quant

A production-ready, 24/7 automated short-term quantitative trading system specifically built for the cryptocurrency market. Designed with an algorithmic mindset, this system runs seamlessly on overseas cloud servers to capture high-frequency alphas and short-term market inefficiencies without human intervention.

## 🚀 Overview
The crypto market never sleeps. This system replaces psychological trading with strict, cloud-deployed algorithmic logic. By utilizing advanced prompt engineering for strategy generation, the system connects directly to mainstream exchange APIs to execute rapid market monitoring, automated risk management, and order routing.

## ⚙️ Key Architectural Components
*   **Strategy Logic Generator**: Leverages LLM terminal orchestration to prototype and refine short-term trading signals.
*   **24/7 Cloud DevOps Pipeline**: Fully deployed on high-availability cloud VPS (Ubuntu/Linux) ensuring uninterrupted uptime and low-latency execution.
*   **API Execution Engine**: Smoothly integrated with CEX API endpoints for real-time order execution, order-book polling, and state tracking.
*   **Risk Mitigation & Validation**: Validated via months of rigorous paper trading simulation to ensure execution stability before deploying live capital.

## 🛠 Tech Stack
*   **Core Language**: Python
*   **Infrastructure**: Cloud VPS (Linux), 24/7 Cron/Daemon Automation
*   **Integrations**: Exchange REST/Websocket APIs
*   **Strategy Engineering**: Quant Indicators (MACD, RSI, Volatility Breakdown), LLM-Assisted Code Optimization

## 📂 Project Structure (Planned)
*   `/bot`: Main trading execution loop and API connectors.
*   `/strategies`: Python scripts for technical indicator parsing and signal triggers.
*   `/config`: Environment setup and API connection wrappers (Secrets omitted).
